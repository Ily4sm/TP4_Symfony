# TP4_Symfony
Ce travail pratique a permis de mettre en œuvre les formulaires avec le framework Symfony en respectant l’architecture MVC et les bonnes pratiques recommandées. Les deux fonctionnalités principales realisees sont l’ajout d’un cours à une wishlist à l’aide d’un formulaire sécurisé par un token CSRF, ainsi que l’intégration d’un formulaire de souscription à une newsletter affiché sur l’ensemble du site via le layout principal. La création de FormType dédiés, la gestion des formulaires dans les contrôleurs et leur affichage avec Twig ont permis de mieux comprendre la séparation entre l’affichage et le traitement des données. L’utilisation d’une route unique pour le traitement du formulaire de souscription et la protection CSRF intégrée garantissent la sécurité et la cohérence de l’application. Ce TP a ainsi consolidé la compréhension du fonctionnement des formulaires Symfony et de leur intégration dans une application web structurée.
